# strudel-producer-space-samples

Samples from <https://producerspace.com/> ready to be used with [strudel.cc](https://strudel.cc/).

Import with `samples('github:neshanjo/strudel-producer-space-samples')`.

| Name | Description |
| ---- | ----------- |
| `computer` | Spoken Words from Computer World |
| `dvs1` | Dance Vocal Shouts 1 |
| `dvs2` | Dance Vocal Shouts 2 |
| `hollywood` | Spoken Words from Hollywood |
| `scifi` | Spoken Words from Science Fiction |
| `shv_ab_124`, `shv_c_126`, `shv_e_125`, `shv_e_128`, `shv_f_124`, `shv_f#_122` | Soulful House Vocals, with key and bpm in name. All minor keys except `shv_e_128` which is E major. |
| `sq` | Spoken Quotes |
| `whisper` | Whispered Words |
